database.get_images(folder_name)[source]¶

Recursively read all folder content, open all images with PIL and create array containing images data.

Parameters

folder_name (string) – name of the folder containing the images

Returns

  • batches_data (array) – Nx(224x224) array containing resized images in a normalized float32 format

  • batches_cat (array) – Nx1 array containing the category of each sample as string

  • batches_file (array) – Nx1 array containing the file path of the image sample

  • batches_folder (array) – Nx1 array containing the folder type, can either be ‘test’, ‘train’ or ‘valid’
